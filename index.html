Benchmarking machine

To allow for direct GPU access and avoid performance fluctuations due to the use of a shared host, benchmarks are run on a dedicated server with the following hardware:

CPUIntel Core i5-12400
RAM16 GB (2×8 GB DDR4-3000 CL16)
SSDSamsung 980 500 GB (M.2)
GPU 1 (integrated)🔵 Intel UHD Graphics 730
GPU 2 (dedicated)🔴 AMD Radeon RX 6500 XT
GPU 3 (dedicated)🟢 NVIDIA GeForce GTX 1650
Operating systemUbuntu Server 22.04 64-bit

All core and memory clocks are kept at stock frequencies. The OS is kept in a default configuration as closely as posible. The machine is not running any other tasks in parallel – this page is served from GitHub Pages.

Benchmarks that make use of the GPU are run 3 times (once on each GPU). Benchmarks that don't make use of the GPU are run only once.
